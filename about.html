About Me

My journey as a web developer started with learning the basics of HTML, CSS, and JavaScript to build simple static websites. As I progressed, I ventured into backend development, working with technologies like Node.js and databases. Over time, I improved my skills by exploring responsive design frameworks like Bootstrap and diving into modern JavaScript frameworks like React. I then created a portfolio to showcase my work and gained real-world experience through internships and freelance projects. Now, I specialize in full-stack development, constantly learning new technologies, contributing to open-source projects, and building complex web applications to refine my expertise.
